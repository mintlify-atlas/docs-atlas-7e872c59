# Welcome to Strat Perpetual DEX

Strat Perpetual DEX is a **decentralized perpetual exchange** built on the Initia blockchain using Move smart contracts. The system features a modular architecture where each contract focuses on a single responsibility, working together to provide comprehensive perpetual trading functionality.

<Note>
  Strat leverages Move's security and expressiveness to deliver a robust, gas-efficient perpetual trading platform with advanced features like dynamic spreads, funding rates, and automated liquidity management.
</Note>

## Core Modules

The Strat Perpetual DEX consists of several interconnected modules:

### Governance (`gov`)
Centralized configuration management for all modules. Handles parameter updates, trading pair registration, and system-wide settings.

### Order Management (`order`)
Primary interface for traders to create, cancel, and modify orders. Manages user collateral during active trades and coordinates with other modules for order execution.

### Position Tracking (`position`)
Maintains all open positions with accurate state tracking. Provides position-related calculations including PnL, leverage, and margin requirements.

### Vault (`vault`)
Collateral and liquidity provider contract. Manages LP tokens, collects fees from trades and liquidations, and handles the economic balance between traders and LPs.

### Oracle Aggregator (`oracle_aggregator`)
Fetches and normalizes price data from multiple oracle sources including Stork. Ensures consistent pricing across the entire system.

### Order Executor (`order_executor`)
Designed for keeper bots to execute pending orders when conditions are met. Handles the automated settlement of trades.

## Economic Model

<Info>
  The vault economics work as follows:
  - **Revenue Sources**: Liquidation fees, opening fees, and negative trader PnL
  - **Positive Net PnL**: When traders profit, LP token price decreases
  - **Negative Net PnL**: When traders lose, LP token price remains stable (LPs profit)
</Info>
