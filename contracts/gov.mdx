---
title: "Governance Module"
description: "Central governance module for managing configuration and protocol settings"
---

The `gov` module serves as the central governance interface for the Strat Perpetual DEX. It manages all configuration updates, pair listings, oracle settings, and protocol-level administrative functions.

## Module Structure

```move
struct ModuleStore has key {
    gov_address: address,
    protocol_fee_address: address,
    order_executor_capability: OrderExecutorCapability,
    order_capability: OrderCapability,
    oracle_aggregator_capability: OACapability,
    pair_capability: PairCapability,
    position_capability: PositionCapability,
    vault_capability: VaultCapability,
    referral_capability: ReferralCapability
}
```

The governance module stores capabilities for all protocol modules, enabling centralized administration.

## Order Configuration

Updates global order module configuration parameters.

```move
public entry fun update_order_config(
    account: &signer,
```

</ParamField>

</ParamField>

<ParamField path="wind_down_window" type="u64" required>

</ParamField>

## Pair Management

### register_pair

Registers a new trading pair with comprehensive configuration.

```move
public entry fun register_pair(
    account: &signer,
    asset_id: u64,
    group_name: String,
    collateral_metadata: Object<Metadata>,
    // pair configs
    open_interest_cap: BigDecimal,
    allow_extended_hours: bool,
    max_leverage: BigDecimal,
    max_overnight_leverage: BigDecimal,
    min_position_value: BigDecimal,
    maker_fee_rate: BigDecimal,
    taker_fee_rate: BigDecimal,
    vault_fee_rate: BigDecimal,
    // funding fee configs
    hill_inflection_point_magnitude: BigDecimal,
    hill_inflection_point_is_positive: bool,
    max_funding_fee_rate_per_second: BigDecimal,
    base_spring_factor: BigDecimal,
    spring_factor_up_scale: BigDecimal,
    spring_factor_down_scale: BigDecimal,
    hill_positive_scale: BigDecimal,
    hill_negative_scale: BigDecimal,
    // rollover fee configs
    max_rollover_fee_rate_per_second: BigDecimal,
    broker_premium_rate_per_second: BigDecimal,
    is_negative_rollover_allowed: bool,
    // dynamic spread configs
    max_spread_ratio: BigDecimal,
    spread_volume_threshold: BigDecimal,
    spread_decay_rate: BigDecimal
)
```

<ParamField path="account" type="&signer" required>

</ParamField>

<ParamField path="collateral_metadata" type="Object<Metadata>" required>
  Metadata object for the collateral token
</ParamField>

<ParamField path="open_interest_cap" type="BigDecimal" required>
  Maximum total open interest allowed for this pair

<ParamField path="allow_extended_hours" type="bool" required>

<ParamField path="min_position_value" type="BigDecimal" required>

</ParamField>

<ParamField path="maker_fee_rate" type="BigDecimal" required>

<ParamField path="taker_fee_rate" type="BigDecimal" required>
