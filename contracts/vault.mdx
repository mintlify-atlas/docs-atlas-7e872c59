---
title: "Vault Module"
description: "Liquidity provider system with deposit/withdraw mechanics and LP token management"
---

The `vault` module manages the liquidity provider (LP) system, handling deposits, withdrawals, LP token minting/burning, and vault PnL distribution.

## Core Data Structures

### LiquidityBalance

Represents a user's LP balance.

<ResponseField name="collateral_metadata" type="Object<Metadata>">
  Metadata object for the collateral token
</ResponseField>

<ResponseField name="amount" type="u64">
  Amount of LP tokens held
</ResponseField>

<ResponseField name="average_price" type="BigDecimal">
  Average LP token price when deposited (for event emission)
</ResponseField>

### LiquidityInfo

Complete liquidity state for a collateral type.

<ResponseField name="collateral_metadata" type="Object<Metadata>">
  Metadata object for the collateral token
</ResponseField>

<ResponseField name="epoch_state" type="EpochState">
  Current epoch state including average open PnL
</ResponseField>

<ResponseField name="max_supply_info" type="MaxSupplyInfo">
  Maximum supply configuration and state
</ResponseField>

<ResponseField name="max_pnl_per_token_info" type="MaxPnlPerTokenInfo">
  PnL per token limits and tracking
</ResponseField>

<ResponseField name="open_pnl" type="SignedBigDecimal">
  Current open PnL in USD value
</ResponseField>

<ResponseField name="open_pnl_snapshot" type="BigDecimal">
  Snapshot of average open PnL
</ResponseField>

<ResponseField name="pnl_per_token" type="SignedBigDecimal">
  Current PnL per LP token (represents liquidity buffer if positive)
</ResponseField>

<ResponseField name="pnl_per_token_snapshot" type="SignedBigDecimal">
  Snapshot of PnL per token at epoch boundary
</ResponseField>

<ResponseField name="last_collateral_price" type="BigDecimal">
  Most recent collateral price
</ResponseField>

<ResponseField name="reward_per_token" type="BigDecimal">
  Accumulated rewards per LP token
</ResponseField>

<ResponseField name="share_price" type="BigDecimal">
  Current LP token share price
</ResponseField>

<ResponseField name="supply" type="u128">
  Total LP token supply
</ResponseField>

### EpochInfo

Vault epoch configuration and timing.

<ResponseField name="epoch" type="u64">
  Current epoch number
</ResponseField>

<ResponseField name="epoch_start_timestamp" type="u64">
  Timestamp when current epoch started
</ResponseField>

<ResponseField name="cooldown_start_timestamp" type="u64">
  Timestamp when withdrawal cooldown period begins
</ResponseField>

<ResponseField name="epoch_end_timestamp" type="u64">
  Timestamp when current epoch ends
</ResponseField>

<ResponseField name="unlock_period" type="u64">
  Number of epochs before withdrawal unlocks
</ResponseField>

<ResponseField name="epoch_duration" type="u64">
  Duration of each epoch in seconds
</ResponseField>

<ResponseField name="cooldown_start_after" type="u64">
  Time after epoch start when cooldown begins
</ResponseField>

<ResponseField name="open_pnl_snapshot_interval" type="u64">
  Interval for taking open PnL snapshots
</ResponseField>

### WithdrawRequestResponse

Withdrawal request details.

<ResponseField name="addr" type="address">
  User address that created the request
</ResponseField>

<ResponseField name="metadata" type="Object<Metadata>">
  Collateral token metadata
</ResponseField>

<ResponseField name="unlock_epoch" type="u64">
  Epoch when withdrawal becomes available
</ResponseField>

<ResponseField name="auto_amount" type="u64">
  Amount that can be auto-withdrawn by anyone
</ResponseField>

<ResponseField name="manual_amount" type="u64">
  Amount that can only be withdrawn by owner
</ResponseField>

### CollateralInfo

Summary information for a collateral type.

<ResponseField name="collateral_metadata" type="Object<Metadata>">
  Metadata object for the collateral token
</ResponseField>

<ResponseField name="current_supply" type="u128">
  Current LP token supply
</ResponseField>

<ResponseField name="max_supply" type="u128">
  Maximum allowed LP token supply
</ResponseField>

<ResponseField name="share_price" type="BigDecimal">
  Current LP token price
</ResponseField>

<ResponseField name="epoch_state" type="EpochState">
  Current epoch state
</ResponseField>
